# TELECOM-CHURN-ANALYSIS
Customer churn analysis aims to identify the factors that influence customers to discontinue a company's services. In this project, the telecom customer dataset is analyzed using Python to explore customer demographics, service usage, contract types, payment methods, and tenure. Through data cleaning, exploratory data analysis (EDA), key performance indicators (KPIs), and visualizations, the project uncovers patterns associated with customer churn. The insights generated help businesses understand customer behavior and develop data-driven strategies to improve customer retention and reduce churn.
